# External Resources and Integrations

## API Integrations

### Search Engines
- **Google Custom Search API**
  - Endpoint: `https://www.googleapis.com/customsearch/v1`
  - Rate Limit: 100 queries/day (free tier)
  - Documentation: https://developers.google.com/custom-search/v1/introduction
  
- **Bing Search API**
  - Endpoint: `https://api.bing.microsoft.com/v7.0/search`
  - Rate Limit: 1000 queries/month (free tier)
  - Documentation: https://docs.microsoft.com/en-us/bing/search-apis/

- **DuckDuckGo Instant Answer API**
  - Endpoint: `https://api.duckduckgo.com/`
  - Rate Limit: No official limit
  - Documentation: https://duckduckgo.com/api

### Content Analysis APIs
- **OpenAI GPT API**
  - Endpoint: `https://api.openai.com/v1/chat/completions`
  - Use Case: Content summarization, classification
  - Documentation: https://platform.openai.com/docs/api-reference

- **Google Cloud Natural Language API**
  - Endpoint: `https://language.googleapis.com/v1/documents`
  - Use Case: Entity recognition, sentiment analysis
  - Documentation: https://cloud.google.com/natural-language/docs

- **Amazon Comprehend**
  - Endpoint: AWS Comprehend service
  - Use Case: Text analysis and insights
  - Documentation: https://docs.aws.amazon.com/comprehend/

### Data Storage APIs
